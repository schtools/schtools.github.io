Online Unblocker Tool

This is a modification of the window that would normally show up when you would try to access an unclassified site.
This version allows you to change the URL you want to unblock and the URL of the Contentkeeper Appliance.

About

The inspiration for the tools I have made is simply because I became annoyed that my institution had blocked some useful resources such as The Wayback Machine.
I have decided to make this website to share these tools with the world for others that may have even stricter content blockers.
These tools were developed for a Contentkeeper system that is only installed on the network. These tools may not work for everyone or every site.

I am not responsible for any trouble I may get anyone into if they choose to use these tools, this is on you pal. Furthermore these tools are for educational use only, don't use them maliciously!!!
